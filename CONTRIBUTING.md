# Contributing

Thank you for your interest in this project! 

## Project Status
This repository is primarily a **learning portfolio** and coursework archive for the Microprocessors and Microcontrollers Lab (CE470). As such, it is not actively seeking feature contributions or major refactors.

## How to Contribute
While this is a personal learning project, I welcome:
- **Bug Reports**: If you find an error in the logic or comments.
- **Suggestions**: Ideas for optimization or alternative implementations are appreciated for educational purposes.
- **Questions**: Feel free to open an issue if you have questions about the code or concepts.

## Code of Conduct
Please ensure all interactions are respectful and constructive.
